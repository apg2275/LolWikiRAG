Ultra Rapid Fire returns from October 1st, 2020 to November 9th, 2020. Clash The second round of Worlds Clash is live, with a 16-team bracket and special rewards. League Client Bug Fix: Fixed an issue where the in-game settings menu was not loading if a game had not been played since the previous patch. Bug Fix: Debug text will no longer appear during promotion ceremonies. Bug Fix: Fixed an issue where gift notifications would appear even after being acknowledged or dismissed. Bug Fix: League taskbar icon should now light up when notifications/messages are pending. Bug Fix: Restoring Default Hotkeys out of game no longer unbinds "Area Is Warded Ping" in the Options Menu. Mac Bug Fix: Fixed an issue where Mac players would see unusual fullscreen brightness when certain champions would appear on screen. Ranked The 2020 ranked season ends on November 9, 2020 (11:59 PM PT). Your Shop Your Shop returns from October 1st, 2020 to November 9th, 2020 (18:00 CET). Game [ ] Emotes Bug Fix: Changed how emotes are sized to make them more consistent across various resolutions and have fixed positions relative to the health bar. Added an option in the in-game settings to change the size of emotes to "Normal" or "Small", so players can choose how they appear.
