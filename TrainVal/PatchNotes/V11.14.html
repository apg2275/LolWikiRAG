League of Legends V11.14[]

Client[]

  • Rotating Game Mode: Ultimate Spellbook arrives to live servers from July 8, 2021 (1:00 AM PT) to August 9, 2021 (11:59 PM PT).
Chat restrictions
  • New chat restrictions will be issued in 3- and 7-day increments, rather than 10- and 25-game increments.
League Client
  • Bug Fix: Fixed a bug where new players who first log in get a popup window telling them that their settings were unable to be retrieved from the server.
  • Bug Fix: Fixed a memory leak with Champion Mastery tooltips.
  • Bug Fix: Banners now properly display in party lobbies.
  • Bug Fix: Fixed a bug where if a player is in a spectator slot in a custom game lobby when the lobby is disbanded, they are taken to a black home screen.

Champions[]

Akali
  • Five Point Strike
    • Removed: Can no longer be cast during Shuriken Flip's recast.
      • Attempting to cast the ability during either dash will buffer it to cast as soon as the dash ends.
  • Perfect Execution
    • Second cast minimum base damage reduced to 60 / 130 / 200 from 75 / 145 / 215.
      • Second cast maximum base damage reduced to 180 / 390 / 600 from 225 / 435 / 645.
